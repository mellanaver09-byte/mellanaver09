## Projects

###  [Spring Acidification Regression Model](./spring-acidification)
Built a regression model on Florida spring water quality data to identify 
key environmental drivers of acidification trends.
**Tech:** Python, pandas, scikit-learn

###  [Soil Property Classification with Terrain Data](./soil-classification)
Classified soil organic matter, cation exchange capacity, and texture 
using Random Forest and SVM models trained on DEM-derived terrain 
covariates (slope, aspect, TWI).
**Tech:** R, terra, sf, sp, raster, Random Forest, SVM
